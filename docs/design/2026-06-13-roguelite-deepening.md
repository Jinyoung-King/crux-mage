# 크럭스메이지 로그라이트 심화 설계

작성 2026-06-13. 현재 v0.68 기준. "성장은 느껴지나 플레이가 안 바뀐다" 해결이 목표.

## 0. 현재 이미 있는 것 (재활용 — 새로 안 만듦)
- **오행 5속성**(목·화·토·금·수) + 상극 매치업(적 상대 ×1.5/×0.7) — `ElementLib`
- **다중 스킬**(v0.65): `player.skills` 목록, 카드로 스킬 획득, 각자 독립 쿨타임. ★심화의 핵심 토대★
- **스킬 단일 처리점** `_skill_hit` / `fire_skill_bolt` — 여기에 행동을 얹으면 전 스킬에 적용됨
- **유물**(런 장착): 즉사·연쇄·점화·재생·황금·격노
- **리스크-리턴 카드** 일부 존재(글래스캐논·속사포·피의계약)
- **보스**(3·5·7·10) + 보너스 코인 웨이브(끝자리8) + 엘리트 수식어(무한)
- **메타**: 캐릭터별 강화·숙련·코인, 무한 모드(HP 무한↑ / 피해 ×12 상한)

→ 즉 "원소·다중스킬·처리점·유물·보스" 골격은 이미 있다. **부족한 건 '깊이'를 만드는 3축: 행동 카드 · 스킬 진화 · 원소 반응.**

## 1. 현재 구조의 문제점
1. **카드가 스탯 막대기**: 공격력+5, 연사+0.5, 스킬위력+35%, 방어+2… 대부분 숫자만 올림. 플레이 방식이 안 바뀐다(=사용자 불만 그대로).
2. **항상-정답 카드**: 공격력/쿨타임은 어떤 빌드든 좋음 → 고민 없이 집음(목표2 위반). 진짜 분기 선택이 없다.
3. **스킬이 정적**: 다중 스킬은 생겼지만 같은 스킬을 또 얻으면 그냥 인스턴스 추가일 뿐 **진화·완성 아크가 없음**(목표3 약함).
4. **원소가 적 상대로만 작동**: 내 스킬끼리의 조합/반응이 없음 → 멀티원소를 굴릴 이유가 약함(목표5 약함).
5. **희귀도가 그냥 큰 스탯**: 전설이 규칙을 안 바꿈.
6. **웨이브가 반복적**: 보스 외엔 중간 목표가 얇음(목표4 약함).

## 2. 지루함의 원인 (한 줄)
**"카드를 골라도 같은 방식으로 더 세게 칠 뿐"** — 선택이 빌드를 *분기*시키지 않고 *증폭*만 한다. 분기와 발견(조합)이 없으면 N판이 다 똑같다.

## 3. 핵심 게임 루프 개선
현재: 웨이브 클리어 → 카드 3장 1택(증폭) → 반복.
개선: **빌드를 분기시키는 선택 + 발견 + 완성**을 루프에 박는다.
- 카드는 기본적으로 **행동 변경**(관통·연쇄·장판·추적·처치폭발…)을 준다.
- 같은 계열을 모으면 **스킬이 진화**(완성 목표).
- 서로 다른 원소 스킬을 섞으면 **반응**(상태부여→격발)이 터진다(발견·조합).
- 중간중간 **이벤트**(상점·제물·균열·선택)로 빌드를 비트는 분기점을 준다.
- 무한 모드는 **상승 난이도 계층(ascension)**으로 반복 동기.

## 4. 카드 시스템 개선 (행동 우선)
### 4.1 모델
스킬 인스턴스(`skills[i]` dict)에 **행동 플래그/수치** 필드를 추가한다: `pierce, chains, homing, split, ground(장판), knockback, on_kill_explode, status_apply, detonate…`. 카드는 이 플래그를 켜거나 키운다. `_skill_hit`/발사체 해석부에서 플래그를 읽어 처리.
→ 구현 난이도 낮음(플래그 분기), **조합 수는 스킬×플래그로 폭증**(목표5).

### 4.2 카드 타겟팅
카드는 **특정 스킬/원소/전역** 중 하나에 붙는다:
- "선택한 스킬에 관통 부여" (드래프트 시 보유 스킬 중 택1)
- "모든 화(火) 스킬이 화상 장판을 남긴다" (원소 전역)
- "처치 시 폭발" (전역 규칙)
→ 같은 카드도 어디에 붙이냐로 빌드가 갈린다(목표1·2).

### 4.3 항상-정답 제거
순수 스탯 카드는 **소수 + 대가 부착**으로만. "공격력 +40%, 대신 쿨 +20%" 식. 무대가 스탯카드는 일반 등급에서 비중 축소.

## 5. 원소 시너지 — 상태부여 / 격발 모델 (NxN 매트릭스보다 솔로 친화)
각 원소 스킬은 **상태**를 적에 부여하고, 다른 원소가 **격발**한다. (PoE식, 구현 단순·조합 풍부)

| 원소 | 부여 상태 | 격발 시(다른 원소가 칠 때) |
|---|---|---|
| 화(火) | 화상(지속 피해) | 폭발 — 주변 광역 |
| 수(水) | 빙결/둔화 | 파쇄 — 빙결 적에 치명 추가타 |
| 금(金, 뇌전) | 감전(표식) | 전도 — 감전 적들끼리 연쇄 |
| 목(木, 비전/자연) | 중독/표식 | 만개 — 중독 중첩 즉시 터뜨림 |
| 토(土) | 균열(방어↓·취약) | 분쇄 — 취약 적에 ×피해 |

**명명 반응(연출/도감용)** — 두 상태가 겹치면 특수:
- 화+금 = **플라즈마**(감전+화상 → 큰 폭발)
- 수+금 = **초전도**(빙결 적 통해 연쇄 강화)
- 화+수 = **증기**(화상+빙결 → 광역 + 시야 연막)
- 토+화 = **용암 장판**, 목+수 = **부패 장판**, 목+금 = **포자 감전**

→ 단일 원소 빌드(깊게) vs 2원소 반응 빌드(터뜨림) 분기 발생(목표1·2·5).

## 6. 스킬 진화 트리 (같은 계열 모으면 상위로)
스킬별 **진화 포인트**(같은 스킬 카드/계열 모디파이어 획득 시 +1). 임계치에서 자동 진화 OR 진화 카드 등장. 분기 가능.

마력탄(목): 마력탄 → 연발 마력탄(3연사) → 유도 마력탄(추적) → **비전 폭풍**(상시 난사)
메테오(화): 메테오 → 쌍둥이 메테오 → 화염 폭풍(장판) → **운석 강우**(전역 연쇄 낙하)
체인(금): 연쇄 → 폭풍 번개(분기 연쇄) → 사슬 폭발 → **천벌**(전 화면 감전→전도)
빙결(수): 서리 → 빙festival(범위↑) → 절대영도(빙결 즉사선) → **영겁의 겨울**(상시 빙결장)
융단폭격(토): 융단 → 집중포화 → 지진(넉백+균열) → **궤도 포격**(거대 단일 폭격)

각 라인 3~4티어 × 분기(예: 메테오는 "장판형" vs "단일 거대형" 갈림). → "이 빌드를 완성한다"는 목표 제공(목표3).

## 7. 전설 카드 20 (플레이 규칙을 바꾸는 수준)
1. **공명**: 스킬 시전 시 보유한 다른 원소 상태를 즉시 격발.
2. **메아리**: 모든 스킬이 0.3초 뒤 한 번 더 발동(50% 위력).
3. **원소 폭주**: 모든 스킬이 매 시전마다 무작위 원소로 변함(상태 떡칠).
4. **시간 왜곡**: 쿨타임 -50%, 대신 적 이동속도 +40%(리스크).
5. **수확자**: 적 처치 시 그 적의 원소 상태를 주변에 전파.
6. **과부하**: 스킬 위력 +120%, 대신 쿨타임 2배(소수 정예 빌드).
7. **분열의 핵**: 모든 투사체/광역이 2회 분열.
8. **저주받은 보고**: 카드 보상 +1장 & 희귀도↑, 대신 적 +20% 강함.
9. **원소술사**: 동시에 3종 이상 원소 보유 시 전 피해 ×1.5.
10. **연성**: 같은 스킬 2개 보유 시 즉시 다음 티어로 진화.
11. **재앙**: 보스/엘리트에게 피해 ×2, 잡몹에게 ×0.6.
12. **부동의 요새**: 기지 내구도 +200%, 대신 회복 불가.
13. **흡열로**: 화상 적 처치 시 쿨타임 즉시 충전.
14. **빙하기**: 빙결 적은 받는 모든 피해 +50%.
15. **뇌神**: 감전 연쇄 무한 점프(사정거리 내), 대신 단일 피해 -40%.
16. **포자 군락**: 중독으로 죽은 적이 포자 구름을 남겨 전염.
17. **천체 정렬**: 10초마다 모든 스킬 동시 발동.
18. **등가교환**: 카드 1장 버릴 때마다 영구 +스킬 위력(빌드 압축).
19. **무한의 룬**: 유물 슬롯 +3(런 한정).
20. **종말 시계**: 매 웨이브 점점 강해짐(+위력), 대신 기지 내구도 매 웨이브 -5%.

## 8. 일반/고급 카드 50 (스탯 막대기 금지 — 행동·원소·조합 부품)

→ 50장 중 순수 스탯카드는 사실상 0. 전부 **행동·조합·대가**. 카드×스킬×원소×붙이는 위치 = 실효 빌드 수백.

## 9. 중간 이벤트 (반복 금지)
- **3 미니보스 / 5·10 보스**(있음, 유지)
- **상점 웨이브**(예: 끝자리 4): 코인으로 카드/리롤/유물 구매
- **제물 제단**(끝자리 7): 기지 내구도 or 카드 1장을 바쳐 희귀+ 보상
- **원소 균열**(끝자리 9): 한 원소를 골라 그 원소 스킬 강화 + 그 원소 상태 격발 강화
- **엘리트 웨이브**(끝자리 6): 변종 1마리 + 확정 희귀 카드
- **갈림길**(보스 후): "안전(회복) vs 도전(적 강화 대신 전설 확정)" 2지선다
→ 다음 목표를 항상 기대(목표4).

## 10. 장기 플레이 동기
- **상승 계층(Ascension)**: 클리어할수록 더 어려운 변형 규칙 해금(적 추가 능력·시작 디버프) → 같은 빌드도 다르게 굴려야 함.
- **메타 해금**: 플레이로 **새 스킬·카드·원소 반응·캐릭터·유물** 해금(현재 캐릭터 해금 구조 확장). 콘텐츠가 잠겨있다 열리는 진행감.
- **원소/스킬 도감**: 발견한 반응·진화 기록 → 수집 동기.
- **시드/일일 도전**: 고정 시드 랭킹(솔로는 로컬 기록으로 시작).
- **빌드 다양성 보상**: 특정 빌드 완성 시 메타 보너스(예: 순수 화염 50웨이브 → 신규 전설 해금).

## 11. MVP (솔로가 먼저 박을 것 — 최대 효과/최소 노력)
기존 `skills` dict + `_skill_hit`/발사체 처리점을 그대로 활용하므로 신규 시스템이 적다.
1. **행동 플래그를 스킬에 추가** + 그걸 켜는 **행동 카드 12~15장**(§8의 관통/연쇄/장판/처치폭발/유도/분열 등). → 즉시 "플레이가 바뀐다" 체감. **최우선.**
2. **상태부여→격발 1세트**: 화상(부여)+감전(격발) 정도만 먼저. 멀티원소 굴릴 이유 생김.
3. **스킬 진화 1~2라인**(예: 마력탄·메테오, 3티어): 같은 스킬/조각 모으면 진화. "완성" 맛.
4. **상점 웨이브 1종**: 코인 사용처 + 선택 고민.
5. 순수 스탯카드 비중 축소 + 대가 부착.
→ 이 5개만으로 목표 1·2·3·5 즉시 개선. 전설·전체50장·전 이벤트는 이후 단계.
