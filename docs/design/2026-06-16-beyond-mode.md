# 저편(Beyond) 모드 — 엔드게임 3축 설계

작성 2026-06-16, v3.11 기준. 목표: 기존 두 모드와 **차별되는** 베테랑용 콘텐츠. 논의에서 나온 세 불만 직격 — ① 매번 비슷함(다양성) ② 목표 흐릿(도착지) ③ 쌓이는 맛 약함.

## 0. 이미 있는 것 (재활용 — 새로 안 만듦)
- `GameState.game_mode`: `"endless"`(무한 순환·기록 경쟁) / `"stage"`(단일 속성 12웨이브 유한 클리어). → 저편은 세 번째 값 `"beyond"`.
- 스테이지 스캐폴딩 전부 재사용: `_wave_kind` / `_stage_element` / `_boss_wave_for` / `_boss_enemy_name` / `_show_boss_banner` / `_stage_spawn_list` / `_stage_cleared` / 결과 화면.
- 엔드게임 축: **무한=기록**, **스테이지=상승(Ascension) 사다리**, **런 내 다양성=중간 이벤트(rift)**. 저편은 이들과 역할이 겹치지 않아야 함.
- 전투: 평타(약한 기본공격) 자동 + 다중 스킬 쿨마다 자동 시전(`player._process`). 평타는 그대로, 저편은 **스킬 자동시전만** 수동으로 전환.

## 1. 저편의 고유 역할 (3축의 차별점)
| 축 | 모드 | 구성 | 조작 | 진행 |
|---|---|---|---|---|
| 기록 | endless | 단일 흐름 무한 | 자동 | 최고 웨이브 |
| 사다리 | stage + 상승 | 단일 속성 12웨이브 | 자동 | 상승 계층 |
| **여정·숙련** | **beyond** | **3장 멀티 속성 여정** | **수동 스킬** | **(P2) 정수 해금** |

저편을 다른 둘과 가르는 4가지:
1. **멀티 속성 챕터 여정** — 스테이지는 단일 속성. 저편은 3개 장이 각기 다른 속성(매 런 5속성 중 3개 무작위 → 매번 다른 여정). → "매번 비슷함" 해결.
2. **수동 스킬 조준·발동** — 평타 자동 유지, 스킬만 탭으로 직접 조준·시전. → 차별성·실력 표현, "조작 없냐"는 반복 피드백 해소.
3. **웨이브 30 게이트** — 최고 웨이브 ≥ 30 도달 시 해금(이미 저장되는 `best_wave` 재활용). → 베테랑 전용, 캐주얼 홈 비노출(접근성 무손상).
4. **(Phase 2) 정수 재화 + 콘텐츠 해금 상점** — 스탯이 아닌 *플레이가 바뀌는* 해금 위주. 상승의 코인 보너스와 역할 분리(중복 금지).

## 2. 구조 (Phase 1)
- 12웨이브 = **3장 × 4웨이브**(일반 3 + 장 보스 1). 장 경계 = index 3·7·11에 보스.
- 장마다 속성 전환(`_stage_element`가 `beyond`에서 장별 속성 반환). 매 런 시작 시 5속성 중 3개를 골라 순서 고정.
- 마지막(3장) 보스 격파 = **"저편 클리어"** — 끝이 있는 관문(무한과 또 다른 차별점).
- 난이도: 무한 ~25~30층 상당 베이스라인(`endless_*_scale`에 고정 레벨 주입). 베테랑용.
- 진입: 홈에 '저편' 버튼. 미해금이면 잠금 + "최고 웨이브 30 도달 시" 안내.

## 3. 빌드 순서 (점진) — "느낌 먼저"
- **v3.12** — beyond 골격 + 멀티속성 여정 + 클리어 + **수동 스킬 조준·발동** + **인게임 드래프트 제거(기본 로드아웃)**. 첫 드롭부터 조작·구조로 뚜렷이 차별.
  - 수동: 저편에서 `player._process` 스킬 자동시전 차단 → 스킬 아이콘 탭으로 무장 → 전장 탭 지점에 시전(`execute`에 aim 좌표 추가, 군집 자동조준 대체). 평타는 자동 유지.
  - 무드래프트: `_ready`가 `_open_draft` 생략 + `_on_wave_cleared`가 상점/이벤트/드래프트 생략 → 바로 다음 웨이브(장 경계 배너).
- **v3.13** — 정수 재화 + **로드아웃·강화 화면**: 보유 스킬을 정수로 진화(`EVOLVE_BRANCHES`: 화상·둔화·관통·장판·처치폭발·표적+) 사전 장착. *수치가 아니라 행동이 바뀌는* 강화 → "쌓이는 맛" + 함정 회피. 저편 정체성 완성.
- **Phase 3** — 장별 전용 신규 적·바이옴 아트, 장 4~5 확장, '여정' 지도 화면.

## 4. 가드레일
- 새 재화는 **콘텐츠·빌드 해금 위주**(스탯 막대 X) — "성장하나 플레이 안 바뀜" 재발 방지.
- **엔드게임 모드 난립 주의**: 이미 스테이지+상승이 유한 엔드게임. 저편은 *수동 조작 + 멀티 여정*이라는 뚜렷한 차별이 있을 때만 정당화됨. 차별이 흐려지면 통합 검토.
- 캐주얼 접근성: 저편은 게이트 뒤 → 신규 유저 홈에 비노출.
